# Proyecto 2 - Predicción de Retorno de Clientes

## Descripción
Este proyecto utiliza técnicas de ciencia de datos y machine learning para predecir si un cliente regresará el siguiente mes.

## Dataset
El conjunto de datos incluye información sobre:
- edad
- género
- gasto total
- frecuencia de compra
- interacción con marketing

## Modelos utilizados
- Regresión Logística
- Árbol de Decisión
- Random Forest
